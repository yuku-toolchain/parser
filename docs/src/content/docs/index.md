<h1 style="font-size: 5rem;margin-bottom: 2rem;color: var(--sl-color-accent);">yuku</h1>

A high-performance JavaScript/TypeScript toolchain written in Zig, bringing modern JavaScript tooling infrastructure to the Zig ecosystem.

<br />

</div>

:::note
Yuku is still at a very early stage. Even though Yuku's JavaScript parser is already functional, JSX and TypeScript support are still in development.
:::

## Roadmap

- [x] JavaScript Parser
- [x] WASM (try here: https://yuku-parser.vercel.app/playground)
- [ ] JSX Support
- [ ] TypeScript Support
- [ ] Best-in-class Visitor and Traverser
- [ ] Node (NAPI) Bindings
- [ ] Transpiler
